# ImageToEditablePptx

一个 Qt Widgets 小工具：选择一张图片，按指定行列拆分，并导出为 `.pptx`。导出的 PPTX 中，每个图片块都是一个独立图片元素，可以在 PowerPoint/WPS 中单独选中、移动、缩放、裁剪和调整层级。

> 注意：原始图片是位图时，工具不会把图片内容自动识别成文本、表格或矢量形状；这里的“可编辑单元”指 PPT 中的独立图片块对象。


项目支持 Qt 5 或 Qt 6，需要安装 `Widgets` 和 `Gui` 模块。

## 使用

1. 打开程序，点击“选择图片”。
2. 设置行数和列数，左侧会预览拆分网格。
3. 点击“导出 PPTX”。
4. 在 PowerPoint/WPS 中打开导出的文件，每个块都可以单独编辑。
